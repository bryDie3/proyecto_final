# proyecto_final# Estructuras de Datos en Python

Este proyecto contiene un único archivo Python que implementa y demuestra las siguientes estructuras y algoritmos:

- Listas simplemente enlazadas
- Listas doblemente enlazadas
- Árbol Binario de Búsqueda (BST)
- Recorrido BFS (Breadth-First Search)

## Requisitos

- Python 3.8 o superior

## Funcionalidades

### Lista Simple
Permite crear, insertar y recorrer nodos en una lista simplemente enlazada.

### Lista Doble
Permite recorrer los nodos tanto hacia adelante como hacia atrás.

### BST (Binary Search Tree)
Implementa un árbol binario de búsqueda para almacenar y organizar datos de manera eficiente.

### BFS (Breadth-First Search)
Realiza un recorrido por niveles sobre el árbol utilizando el algoritmo Breadth-First Search.
